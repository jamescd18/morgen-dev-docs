# Introduction

The Morgen APIs allows developers to build features that 
interact with calendars hosted on different providers, including 
Google, Microsoft, Apple and other CalDAV providers.


## Get your Client ID and Client Secret

To use the Morgen APIs, you need to register your application with
Morgen. Currently, this is a manual process and the APIs are only
available to a limited number of partners. If you are interested in
using the APIs, please contact us at connect@morgen.so.
