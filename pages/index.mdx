# Introduction
Morgen API allows developers to build features that interact with calendars hosted on different providers, including Google, Microsoft, Apple and other CalDAV providers.

## What can I do?
**Morgen API** allows you to:
- Integrate in your software a lot of functionalities available in Morgen. Learn more here: [Morgen API Reference](/sync-introduction).
- Develop [Custom Workflows](/custom-workflows) that interact with calendars and **run on our infrastructure** without you having to deploy anything.



## Get Your API Key
To use the Morgen API, you need to get an API key. You can get one by signing up at [https://platform.morgen.so](https://platform.morgen.so).

Learn more in the [Authentication](/cw-authentication) section.
